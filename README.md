# Muninn（记忆之鸦）

> 在北欧神话中，主神 Odin 拥有两只神鸦：**Huginn（代表思想 thought）** 与 **Muninn（代表记忆 memory）**。它们每天破晓时分飞往人世间收集情报，并在黄昏时飞回，落在 Odin 的肩膀上，低语它们所见所闻的一切。

本项目命名为 **Muninn**，寓意为个人的「数字记忆库」。它记录了关于开发环境配置、命令行工具使用、英语语法辨析以及各种实用指南，作为随时可以检索和查阅的知识沉淀中心。
