# Zellij


[Zellij](https://github.com/zellij-org/zellij)（发音类似于 "Zel-eezh"）是一个用 Rust 编写的现代化终端多路复用器（Terminal Multiplexer）。

```
cargo install --locked zellij
```

简单来说，它的功能类似于老牌的 tmux 或 screen，允许你在同一个终端窗口里切出多个分屏（Panes）、标签页（Tabs），并管理后台会话（Sessions）。但 Zellij 的定位更进一步，它不只是一个多路复用器，而是一个终端工作空间（Workspace）。

## 核心杀手锏

- 零记忆成本：自带“小抄”的状态栏
启动 Zellij 后，终端底部会常驻一个高颜值的动态提示栏。你当前处于什么模式（锁定模式、分屏模式、标签模式），它就会实时把可用的快捷键展示在底部。

- 强大的 WebAssembly (Wasm) 插件生态
Zellij 内部可以用 WebAssembly 运行第三方插件。它自带了一个 Strider（文件浏览器）插件，不需要装任何额外工具，在 Zellij 里按快捷键就能在左侧弹出一个极度丝滑的网页级树状文件目录。

- 智能布局系统 (Layouts)
可以通过简单的配置文件（KDL 格式）定义 Zellij 的“工作皮肤”。比如定义一个 frontend.kdl：一键启动后，上方左侧打开 Helix 写代码，上方右侧实时显示打包日志，下方开一个底层 Shell 跑 Git 命令。每次进入项目，一键就能完美复原整个工作空间。

- 极其丝滑的鼠标支持
Zellij 对鼠标进行了原生的一等公民对待，滚轮滚动、点击、拖拽全都非常符合现代操作系统直觉。


## 运作时的视觉结构

在终端里启动 Zellij 后，它通常会把你的屏幕划分为三层：

```
+-------------------------------------------------------------+
| [Tab 1: Code]  [Tab 2: Server]  [Tab 3: Git]  <-- 顶部标签栏  |
+-------------------------------------------------------------+
|                  |                                          |
|  左分屏 (Pane)    |            右分屏 (Pane)                  |
|  (例如：写代码)    |         (例如：跑测试/看日志)                |
|                  |                                          |
+-------------------------------------------------------------+
|  Ctrl + p: Pane  |  Ctrl + t: Tab  |  Ctrl + n: Resize ...  |
|  <-- 底部“小抄”状态栏（会根据你按下的键实时切换提示）                |
+-------------------------------------------------------------+
```

## 组合 Alacritty / Ghostty + Zellij + Helix

1. Alacritty / Ghostty 提供最底层的 GPU 硬件加速 与超低延迟的窗口渲染。
2. Zellij 在终端里接管一切，负责开多窗口、多标签、管理后台进程，并提供树状文件浏览器。
3. Helix 嵌套在 Zellij 的主分屏里，负责纯键盘高频编辑代码、通过 LSP 补全。
