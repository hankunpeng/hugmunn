```shell
# https://ghostty.org/download
# or
brew install --cask ghostty
```

## Shortcuts

### 切换标签页（Tab）

快捷键根据操作系统有所不同，macOS 的 cmd 键对应 Windows / Linux 的 ctrl 键。

- 切换到下一张标签页 (Next Tab)
  - `cmd + shift + ]`
  - `ctrl + tab`
  - `ctrl + shift + →`
- 切换到上一张标签页 (Previous Tab)
  - `cmd + shift + [`
  - `ctrl + shift + tab`
  - `ctrl + shift + ←`
- 切换到指定标签页
  - `cmd + 1` 至 `cmd + 8`（对应第 1 到 8 个标签页）
  - `cmd + 9`（直接跳转到最后一个标签页）
