# MarkEdit


## 配置别名

```
echo "alias om='open -a MarkEdit'" >> ~/.zshrc && source ~/.zshrc
```


## 默认程序

将 Markdown (.md) 文件的默认打开方式设置为 MarkEdit，macOS 系统对应步骤：
1. 在 Finder 中找到任何一个 .md 文件。
2. 右键点击或按住 Control 键点击该文件。
3. 在弹出的菜单中选择「显示简介」。
4. 展开「打开方式」菜单。
5. 在下拉列表中选择 MarkEdit。
6. 点击下方的「全部更改…」按钮，确认后即可让电脑上所有的 .md 文件默认使用 MarkEdit 打开。
